## httpd server

### Prerequisites
 * Cmake >= 3.13
 * C++17 / GCC >= 8.0
 * Conan
 * For Unix: Makefile
 * For Windows: Visual Studio 15

### Installation

 * Clone this repository
 * On Unix System `cd install && ./install.sh`
 * On Windows System `start install/install.bat`
 * Project binaries built at `/generated`
