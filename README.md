## httpd server

### Prerequisites
 * Cmake >= 3.13
 * C++17
 * gcc recommended >= 8.0
 * Conan
 * For Unix: Makefile
 * For Windows: Visual Studio 15

### Installation

 * Clone this repository
 * On Unix System `./install/install.sh`
 * On Windows System `/install/install.bat`
 * Project binaries built at `/generated`
